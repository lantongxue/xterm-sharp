# Attribution and compatibility baseline

XtermSharp is a clean C# translation and redesign derived from the MIT-licensed
[xterm.js](https://github.com/xtermjs/xterm.js) project.

The compatibility baseline bundled in this workspace is:

- xterm.js version: `6.0.0`
- Git commit: `b1aee19ac`
- Commit date: 2026-07-13

The original xterm.js copyright and MIT license are preserved in [LICENSE](LICENSE).
The `xterm.js/` directory is a development reference and is not included in the
XtermSharp NuGet package.

The Unicode 15.0 data files under `tools/unicode/15.0.0/` are provided by Unicode, Inc. and are
distributed under the Unicode Data Files and Software License included in that directory.
